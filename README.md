# Apresentação do Projeto – Site Responsivo com Bootstrap

O projeto consiste no desenvolvimento de um site totalmente responsivo utilizando o framework Bootstrap. O objetivo é demonstrar como componentes pré-construídos e o sistema de grid da ferramenta podem acelerar o desenvolvimento e garantir consistência visual em diferentes dispositivos.

Principais Recursos Utilizados

Sistema de Grid (Grid System):
Utilizado para organizar o layout em colunas, permitindo que o conteúdo se adapte automaticamente a telas de celular, tablet e desktop.

Componentes Prontos:
Botões, cards, navbar, formulários e carrossel foram implementados para criar uma interface moderna sem necessidade de estilização complexa.

Utilização de Classes Utilitárias:
Classes como mt-3, text-center, d-flex, entre outras, foram aplicadas para ajustes rápidos de espaçamento, alinhamento e responsividade.

Customização com CSS:
Apesar dos padrões do Bootstrap, foram adicionados estilos personalizados para criar uma identidade visual própria ao projeto.

Resultados

Site visualmente limpo e profissional.

Navegação fluida em qualquer dispositivo.

Redução significativa no tempo de desenvolvimento.
